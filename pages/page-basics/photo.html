photoshop界面设置

新建设置

文件 -> 新建 -> width: 1920像素; height: 2000像素; 分辨率: 72像素; 颜色模式: RGB颜色; 背景颜色: 透明 -> 确定[存储预设]

移动工具设置

移动工具栏 -> 图层 ->ctrl + 左击选中图片 = 会选中那张图片的图层

视图设置

视图 -> 选择 智能参考线，标尺( ctrl + r )

窗口 -> 选择 图层，历史记录，信息， 字符。四大面板

信息面板右边 -> 面板选项 -> 颜色: RGB颜色， 标尺单位: 像素。 勾选文档大小 -> 确定

photoshop基本操作

选取内容

矩形选取框(画好大小先放手) + shift（后放手) =正方形

椭圆选取框(画好大小先放手) + shift（后放手) = 圆形

alt 中心点

圆形选取框 -> shift + m键 -> 继续画选框 -> 切换为矩形选框

套手选区域

常用手法

ctrl + alt + 鼠标滚轮 = 放大图片

ctrl + “+” = 放大图片

空格键 = 移动图片

m = 选框快捷键

ctrl + D = 撤销选框

选择好要裁剪区域 -> 剪裁工具( c快捷键 ) ->会剪裁我们选中的地方 -> 保存，导出
